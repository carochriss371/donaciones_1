# Los Buenos Somos Más - Panel de Transparencia

Dashboard público para mostrar donaciones, inventario y transparencia financiera.

## 🚀 Características

- 📊 Resumen financiero en tiempo real
- 📋 Registro contable detallado
- 📦 Control de inventario
- 🤝 Seguimiento de donaciones
- 🔄 Actualización automática cada hora
- 📱 Diseño responsivo

## 🛠️ Tecnologías

- HTML + Tailwind CSS
- JavaScript Vanilla
- GitHub Actions (Cron job)
- GitHub Pages (Hosting)

## 📦 Instalación

```bash
# Clonar repositorio
git clone https://github.com/tu-usuario/donaciones-gh-pages.git
cd donaciones-gh-pages

# Instalar dependencias
npm install

# Ejecutar actualización manual
npm run fetch
